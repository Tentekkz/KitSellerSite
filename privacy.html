Политика конфиденциальности

KitSeller использует данные только для работы сервиса и обработки файлов, которые пользователь отправляет в бот.

1. Какие данные используются

Сервис может обрабатывать файлы, которые пользователь самостоятельно загружает в Telegram-бот.

2. Для чего нужны данные

Данные используются для формирования накладных, сборочных материалов и Excel-отчётов.

3. Передача третьим лицам

Мы не продаём и не передаём пользовательские файлы третьим лицам для рекламы или маркетинга.

4. Контакт

По вопросам сервиса: @kitseller_bot.
